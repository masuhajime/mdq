# mdq php bot

## install

<!-- aaa -->
```
git clone git@github.com:masuhajime/mdq.git
# get composer.phar
curl -sS https://getcomposer.org/installer | php
# get vendor by composer
php composer.phar install
```

## getting start
```
 # login
 php bin/login_sq_member.php
 
 # run blackjack
 php bin/blackjack.php
```
